Dungenium

Service & UI Planning

Date
January - February, 2016
Platform
Mobile Unity
UI planning
Game User research
Information architecture

Summary

Dungeonium is mobile defense social network game. Before my team was really impressed about Supercell, Clash of Clan. The success of SNG game is incredible powerful.

Problem


Before most social network game that focus on defense have less improving character function and amusement. Characters are only for defensing function.

Process


1. Theoretical Research

1) Motivation

According to Richard Bartle, 60% of players are socializer. They focus on social network interaction than enjoyment. In hence, motivation has to consider on two aspects.

2) Flow status

For game players, engagement is critical factor to keeping and selling point which means we have to design cognitive flow. Cognitive flow is the result of cognitive and emotional status by interaction between difficult and skill.

MDA framework

- Mechanics are rules beyond the game.
- Dynamics are visual element which shows systems and rules.
- Aesthetics is how users feel based on game system.

Here is example of how MDA framework works.

1. Level system is really important factor which effects on whole system and users. Depends on level, user could have opportunities to get new items and characters. That probability is mechanics and quest which matched for this is dynamics. Based on this system, user feel challenge and unexpectedness which is aesthetics.
2. Time perception design

3. Give users both sides of enjoyment: defense and attack This game is basically defense game but when user won the game, we can get opponent players’ monsters as prize.

3. Information Architecture

- User flow should be aligned to mental model - The standards of calculation based on not event but official funds for easy to manage

Solution

1. Tagit allows to search easily based on users’ tag and it increase credibility for usage

User can select specific contents in saved and shared lists on facebook and add their thinking to promote value.
